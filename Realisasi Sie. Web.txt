Seksi Web 
Job Description
a. Bekerja sama dengan seksi sponsorship dan seksi dekdok dalam membuat konten portal web.
b. Memelihara dan meng-update web secara berkala.
c. Bekerja sama dengan seksi materi dalam mengelola sistem perlombaan. 
d. Bekerja sama dengan seksi publikasi dalam publikasi Anforcom melalui web.
e. Membuat dan mengelola sistem pendaftaran.

Realisasi:
a. Membuat dan memelihara web.
b. Membuat sistem pendaftaran lomba.
c. Membuat dan mengelola sistem perlombaan.
d. Bekerja sama dengan seksi publikasi perihal update berita.
e. Bekerja sama dengan seksi sponsorship dan seksi dekdok perihal penambahan logo sponsorship dan media partner.
f. Pemeliharaan berkala terealisasi dengan lancar.

Hambatan:
a. Pembagian job description yang kurang merata sehingga terasa sedikit berat untuk anggota tertentu.
b. Desain web otodidak dari seksi web.
c. Kerjasama dengan seksi materi kurang baik sehingga ada suatu teknis upload file peserta tidak sesuai dengan juklak.
d. Kerjasama dengan seksi dekdok kurang maksimal sehingga telat dalam mengupdate logo media partner.

Saran:
a. Pemilihan anggota seharusnya sedikit tapi efisien, 1 web designer, 1 untuk front-end, 1 untuk back-end, dan 1 admin.
b. Melakukan kumpul seksi web secara berkala.
c. Pembuatan konten website seharusnya dirundingkan dengan seksi acara dan seksi publikasi sehingga tercipta konten yang menarik.
d. Website bisa selesai jauh hari sebelum pendaftaran dibuka dengan alasan memublikasikan acara.
e. Desain web seharusnya dibuat oleh seksi dekdok yang mengerti tentang tampilan website atau desain dibuat oleh seksi web itu sendiri.
f. Meningkatkan kerjasama yang baik dengan seksi lain agar tidak terjadi kesalahan pembuatan konten ataupun miskomunikasi.
